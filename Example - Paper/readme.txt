Dataset in this example is in file "database_Number_4.zip".
This dataset is the MNIST dataset, which comprises a collection of handwritten digits. 
Specifically, we consider a subset of 720 images, all showcasing the handwritten digit “four”
Reference:
Y. LeCun, C. Cortes, and C. Burges, “MNIST handwritten digit database,” ATT Labs [Online]. Available: http://yann.lecun.com/exdb/ mnist, vol. 2, 2010
